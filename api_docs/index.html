Lattice Microbes is a software package for efficiently sampling trajectories from the chemical and reaction-diffusion master equations (CME/RDME) on high performance computing (HPC) infrastructure using both exact and approximate methods [ROBE2013], [HALL2014]. pyLM is a problem solving environment written in Python that provides an easy to use and highly customizable simulation construction tools for complex biological scenarios [PETE2013]. jLM is an alternative Python interface to Lattice Microbes designed to integrate with Jupyter notebooks.

For help and general discussion please sign up for the Lattice Microbes mailing list.
