jLM.Lattice.populateLattice()¶

Add particles to lattice at random locations.

The particleDensity argument is a \(N_{siteTypes}\times N_{particleTypes}\) array of slot occupancy probabilities. It describes the probability to find a particle type (including empty==0) at a particular particle slot.

Parameters:
  • pLattice (numpy.ndarray) – Particle lattice (H5 format, shape=(nw,nz,ny,nx,np), dtype=uint8) Modified in place.
  • sLattice (numpy.ndarray) – Site lattice (H5 format, shape=(nz,ny,nx), dtype=uint8)
  • particleDenstiy (numpy.ndarray) – Particle probability, (shape=(Nsites,Nsps), dtype=float64)
Keyword Arguments:
 
  • particleVolFrac (numpy.ndarray) – Array of particle volume fractions indexed by particle lattice species If None, do not consider excluded volume
  • evLattice (numpy.ndarray) – Background excluded volume. If None, assumed to be zero.
  • exact (bool) – If True, make a second pass that corrects the particle counts exactly
  • mask (numpy.ndarray) – Boolean mask to set region of lattice to populate, None (default) populates the entire lattice
  • seed (int) – Seed to RNG. Defaults to time().
