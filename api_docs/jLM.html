jLM is an alternate interface to Lattice Microbes through the Jupyter notebook. For a quick introduction to jLM, please see the example notebook.

jLM jLM top level.
jLM.RDME Spatial model
jLM.RegionBuilder Design lattice geometry
jLM.Solvers Generic RDME solvers
jLM.Types Simulation objects.
jLM.BaseTypes Simulation object base class and collections
jLM.ColorGen Defines ColorSeq, the color scheme generator
jLM.DisplayUtils Auxiliary helper functions for presentation
jLM.Lattice Cythonized lattice manipulation functions
jLM.Template Templating engine tools
jLM.TexFmt Dump simulation parameters as LaTeX
